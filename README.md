# Save the Watermelon 🍉
Joshua Vega

## Description
This is a terminal-based word guessing game. The player must guess the secret word letter by letter. Each wrong guess removes a "slice" of the watermelon. You have 6 slices total to save the watermelon.

## How to Run
1. Go to the src/ folder.
2. Run python game.py.

## Project Structure
docs/: Contains Design, Pseudocode, and Test Plan.
src/: Contains the Python source code.
